YACardEmu GUI — Quick Use Instructions

1. Place YACardEmu_Gui.exe in the same folder as YACardEmu.exe
2. Run YACardEmu_Gui.exe
3. The GUI will automatically read your cards directory from config.ini
4. Select a card from the list
5. Press your assigned Insert key/button or in app Insert button
   to load the card in-game

First Time Use

* Start the game and create/use a card normally
* The GUI will detect new .bin files from the configured cards path
* When prompted, rename the card and choose an image (optional)
* For new cards, the in app preview image will not update text until
  after the next save.

Controls

* Bind a keyboard key or controller button for Insert
* Press a key/button when prompted to assign it

Notes

* Card files are read from the path set in config.ini
* Card images are stored in data\card_images (.png)
* You can add your own images to this folder at any time
* The GUI updates card images after in-game saves

If something doesn’t work

* Verify the cards path in config.ini is correct
* Make sure the game is running before pressing Insert
* Restart the GUI if new cards are not detected immediately
